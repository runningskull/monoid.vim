## Customization

| Setting | |
|-|-|
| `g:monoid_bg_gui = [r, g, b]` | set the background color, from which the other colors derive. |
| `g:monoid_nobold = 1` | disable bold highlights |
| `g:monoid_noitalic = 1` | disable italic highlights |
